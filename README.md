## What this is

A book on the Balanced Scorecard Process (BSCP) — a methodology developed by John G. Tittle for surfacing, measuring, and balancing the competing values an organization is actually operating under. Originally built in outsourcing-consulting work to address the gap between contractually-met service-level agreements and visibly-unsatisfied customers, the methodology has since been used in many client engagements across multiple industries.

The book is presented here as a public, contribute-able work. Readers are welcome to file issues, suggest edits via pull requests, or use the "edit this page" pencil icon at the top of any page on the published site to propose changes directly.

## Companion project

The author's parallel project on spiritual formation, *Intentional Journey of the Heart*, references this book in its Vol 1 supplemental on "Reading the Laws as Held Tensions" — naming BSCP as the operational methodology where the held-tensions / competing-values insight was developed first. See <https://jgtittle-ministries.github.io/Intentional-Journey-of-the-Heart/volume-1-laws-of-the-spirit/supplemental-held-tensions-and-shadow-pairs/> for that cross-reference.
