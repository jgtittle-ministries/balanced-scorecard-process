# Appendix: Metric Options Reference

!!! info "Coming soon"
    This appendix catalogs metric options across the common BSCP categories — commitments, productivity, quality, future value, customer satisfaction, and others — with notes on when each metric is useful and what behaviors it tends to drive in the organization being measured. It is drawn from a 2015 sanitized version of the original appendix that accompanied the methodology in client engagements.

The appendix is structured by category, with each metric definition including:

- Category placement
- Definition and unit of measurement
- Typical algorithm or computation
- Target and threshold setting guidance
- Behaviors the metric tends to drive (intended and unintended)
- When to use; when to avoid
