## Slide 7: 7

![Slide 7](vital-signs-report/slide-007.png)

**Creating the Vital Signs Report**

- Copy Excel file from existing account
- Metric Tabs
- Enter data from Specification Sheet or Summary Matrix
- Change tab name
- Change tab name in macro
- Setup base measures and algorithm
- Add base measure breakdown if needed
- CAPPUD
- Business Unit
- Add scoring model if needed (not piece-wise linear)
- For example, Delivery Rate, Project Budget/Schedule, Survey
- Vital Signs Tab
- Organize rows by categories
- Verify links to Metric Tabs for spark line charts and cells

## Slide 15: 15

![Slide 15](vital-signs-report/slide-015.png)

**Producing the Vital Signs Report**

- Copy from previous month (File+Save as)
- Roll Month with Macro
- Modify each metric tab using desk instructions
- Setup the tab for month
- Roll the tab augmentation area
- Verify formulas and tab specific cells
- Populate with data for month
- Intermediate collection files (validation tools)
- May vary from month to month (Quarterly)
