# Appendices

Supporting reference material for the Balanced Scorecard Process. Each appendix can be read independently of the main chapters and is referenced from the chapters where its content is needed.

## Reference material

- **[Metric Options Reference](metric-options.md)** — A catalog of ~35 fully-specified metrics drawn from production BSCP engagements at three sanitized organizations. Each entry shows the BSCP specification template in detail — definition, business purpose, target, algorithm, base measures with inclusions and exclusions, data flow, aggregation rules, and scoring rules with weights. Use as a reference when defining metrics for a BSCP engagement of your own.

## Training material

Ten training decks developed for delivering the BSCP methodology to consulting teams, plus lessons-learned notes from those deliveries. Each appendix presents the original slides reproduced for historical fidelity alongside the extracted slide text for searchability and accessibility.

- **[Workshops](workshops.md)** — How the BSCP workshop series is structured: the Sponsorship Workshop, the "Where's the Pain?" brainstorming session, category grouping, and the working group dynamics.
- **[GQM (Goal Question Metric)](gqm.md)** — The SEI-derived Goal-Question-Metric process as adapted for BSCP. How to move from felt pain to measurable behavior.
- **[AHP (Analytic Hierarchy Process)](ahp.md)** — Saaty's pairwise-comparison methodology as implemented in Expert Choice, used to weight categories and metrics within the BSCP.
- **[Decision Matrix](decision-matrix.md)** — The BSCP Decision Matrix used to capture and document the working group's decisions as the scorecard takes shape.
- **[Metrics Definition Document](metrics-definition.md)** — How to write the rigorous metric specifications that operationalize each BSCP metric, including algorithm, baseline, scoring, and aggregation rules.
- **[Limits and Scoring](limits-and-scoring.md)** — Setting targets and thresholds for each metric using statistical process control and the piecewise-linear scoring method.
- **[SPC (Statistical Process Control)](spc.md)** — Foundational SPC concepts as applied in BSCP: control charts, baselines, re-baselining triggers, and seismic-event provisions.
- **[Vital Signs Report](vital-signs-report.md)** — The one-page operational deliverable: how it is structured, what each element represents, and how line managers use it week to week.
- **[Implementation Plan](implementation-plan.md)** — The phased implementation approach: T0/T1/T2 phasing, what each phase is responsible for delivering, and how to manage the transition.
- **[Organizational Change Tools](accelerating-change.md)** — The Accelerating Change methodology integrated into BSCP for managing the organizational-behavior transformation the scorecard is designed to drive.
- **[Lessons Learned](lessons-learned.md)** — Observations from delivering the BSCP training series to consulting teams: what worked, what did not, and what to do differently.

Additional appendices may be added as the book is built out. Suggestions are welcome via the [contributing guide](../about/contributing.md).
