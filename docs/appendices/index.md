# Appendices

Supporting reference material for the Balanced Scorecard Process. Each appendix can be read independently of the main chapters and is referenced from the chapters where its content is needed.

- **[Metric Options Reference](metric-options.md)** — *coming soon.* A catalog of metric options across the common BSCP categories (commitments, productivity, quality, future value, etc.), with notes on when each metric is useful and what behaviors it tends to drive. Drawn from the 2015 sanitized appendix to the working drafts.

Additional appendices may be added as the book is built out. Suggestions are welcome via the [contributing guide](../about/contributing.md).
